# Tarea 6 – Gobierno de Datos y Diseño del Proyecto Big Data

## Descripción

Este repositorio contiene el desarrollo de la **Tarea 6**, donde se construyó el Gobierno de Datos y el diseño general del proyecto Big Data para el sistema de monitoreo de calderas. En esta actividad se definieron las políticas, roles, estructuras y componentes necesarios para gestionar los datos generados por las lecturas de temperatura, presión y ppm de cada sensor. También se diseñó la base de datos principal, la hoja de cálculo funcional y el proceso de poblamiento, siguiendo la rúbrica establecida.

Las actividades principales incluyeron:

* Diseño del **Gobierno de Datos** del proyecto, con roles, políticas y reglas de operación.
* Construcción del **Diseño del Proyecto Big Data**, identificando componentes, flujos y dependencias.
* Desarrollo de los **conceptos y elementos de Big Data** aplicados al caso de estudio.
* Diseño y poblamiento de la **hoja de cálculo “lecturas”**.
* Diseño de la **estructura de la tabla “lecturas”** con definición de campos y tipos.
* Creación de la base de datos **monitoreo-produccion**.
* Poblamiento de la base de datos con registros simulados.

## Conclusiones

El desarrollo de esta tarea permitió comprender cómo el Gobierno de Datos influye en la calidad, organización y operación de un proyecto Big Data. La definición de políticas, estructuras y tablas proporcionó una base sólida para el sistema de monitoreo, y el proceso de poblamiento mostró cómo se integran los datos en un entorno funcional. Este trabajo refuerza competencias relacionadas con diseño de sistemas, gestión de información y organización de arquitecturas orientadas a datos.
